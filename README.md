# evolvepro-test

EvolvePro is a cutting‑edge platform designed to streamline project management and accelerate product development. It offers a suite of tools for task tracking, collaboration, and continuous integration, enabling teams to deliver high‑quality software faster and more efficiently.

## Key Features
- **Task Management**: Create, assign, and prioritize tasks with ease.
- **Real‑time Collaboration**: Share updates, comments, and files directly within the platform.
- **CI/CD Integration**: Seamlessly connect with GitHub Actions for automated testing and deployment.
- **Analytics Dashboard**: Monitor progress, velocity, and team performance with intuitive visualizations.

## Getting Started
Clone the repository and install dependencies:

```bash
git clone https://github.com/Chrushingpie/evolvepro-test.git
cd evolvepro-test
npm install
```

Run the development server:

```bash
npm run dev
```

Happy coding!
