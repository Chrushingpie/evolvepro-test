# Contribution Guidelines

Thank you for your interest in contributing to EvolvePro! Please follow these guidelines to help us maintain a high-quality codebase.

1. **Fork the repository** and create a new branch for your feature or bug fix.
2. **Follow the existing code style and formatting conventions**.
3. **Write clear commit messages** that describe the intent of the change.
4. **Submit a pull request** with a descriptive title and a detailed description of the changes.
5. **Ensure all tests pass** before merging.
6. **Include any relevant documentation or screenshots** if applicable.

Happy coding!
